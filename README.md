# CS Student Resources Hub

A comprehensive website containing valuable information and resources for Computer Science students, including learning resources, internship tips, general advice, and more.

## Features

The website is organized into 6 main sections:

1. **Learning Resources** - Online courses, documentation, interactive platforms, and video tutorials
2. **Internship Tips** - Resume building, application strategies, interview preparation, and networking
3. **General Advice** - Study tips, career planning, work-life balance, and common pitfalls
4. **Technical Skills** - Programming languages, development tools, version control, and web development
5. **Interview Preparation** - Data structures & algorithms, system design, coding practice, and behavioral questions
6. **Networking & Community** - Online communities, events & conferences, mentorship programs, and open source

## How to Use

Simply open `index.html` in your web browser to view the website. The navigation menu at the top allows you to jump to different sections.

To view the website:
```bash
# Option 1: Open directly in browser
open index.html

# Option 2: Serve with Python
python3 -m http.server 8000
# Then visit http://localhost:8000/index.html
```

## Structure

- `index.html` - Main website page with all sections
- `styles.css` - Styling and responsive layout

The website is fully responsive and works on both desktop and mobile devices.

## Adding Content

Each section contains empty resource lists ready for links and content to be added. Simply edit the `<ul class="resource-list">` elements in `index.html` to add your resources.
